## 📂 Αρχείο Ασκήσεων

### Άσκηση 1: Soil Moisture Sensor with Arduino
**Ημερομηνία:** 26/02/2026

#### Περιγραφή
Υλοποίηση συστήματος ανίχνευσης υγρασίας εδάφους. Το σύστημα αξιολογεί το επίπεδο υγρασίας και ενημερώνει τον χρήστη μέσω LED και Serial Monitor για το αν το φυτό χρειάζεται πότισμα.

#### Υλικό (Hardware)
* Arduino Board
* Soil Moisture Sensor (Αισθητήρας Υγρασίας)
* 1x Green LED (Pin 6)
* 1x Red LED (Pin 7)

#### Βασικά Σημεία
* **Αναλογική Έσοδος (A0):** Χρήση για ακριβή μέτρηση (0-1023).
* **Ψηφιακή Έξοδος (D0):** Δυνατότητα χρήσης ως διακόπτης βάσει ορίου.
* **Λογική:** Όταν η τιμή είναι κάτω από το όριο (800), το πράσινο LED ανάβει (επαρκής υγρασία). Όταν υπερβεί το όριο, ανάβει το κόκκινο LED (ανάγκη για πότισμα).
